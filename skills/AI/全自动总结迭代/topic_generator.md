# 选题生成器

在碰撞报告产出后运行，将碰撞信号转化为可执行的创作选题。

## Prompt

```
基于以下碰撞信号，生成 1-3 个创作选题建议。

碰撞信号：
{碰撞报告内容}

对每个选题，严格按以下格式输出：

### 选题 N

**标题建议**：{一句话标题，要有信息差或态度，不要平淡描述}
**平台**：小红书 | B站 | 播客BD（三选一）
**形式**：图文 | 短视频 | 长视频 | 专栏 | 对话
**匹配理由**：为什么这个选题适合这个平台？（2句话）
**难度**：轻（30分钟内）| 中（1-2小时）| 重（半天+）
**核心观点**：用一句话说清楚你要传达什么
**角度建议**：从什么角度切入最有力？
**应避免**：这个选题容易踩的坑

## 平台匹配规则

- **小红书**：个人视角、有争议性、短平快、能引发收藏欲
- **B站**：需要演示/可视化、有教程属性、叙事结构完整
- **播客BD**：深度话题、需要对话碰撞、涉及行业判断

## 选题质量规则

- 必须以碰撞信号为基础，不要凭空生成
- 标题必须有"钩子"——让人想点开
- 每个选题必须不同平台，不要重复建议
- 如果碰撞信号太弱，允许输出 0 个选题 + 说明原因
```
