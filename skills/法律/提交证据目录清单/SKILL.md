# 提交证据目录清单

自动从ov文件夹提取案件信息，填充证据目录清单模板。

## 触发条件

用户说"提交证据目录清单"、提供模板docx + 案号信息，要求填充。

## 执行步骤

### 1. 连SMB，定位案件文件夹

```
连SMB → /Volumes/ScannedDocuments/要新建/ov/<案号><申请人>/
```

文件夹内应有：`申请.pdf`、`答复.pdf`、`决定.pdf`

### 2. 提取信息

用pymupdf提取PDF文本层（不必OCR，除非文本层为空）：

```bash
pip3 install pymupdf --break-system-packages  # 如未安装
python3 -c "import pymupdf; doc=pymupdf.open('path'); print(doc[0].get_text())"
```

**从申请.pdf提取：**
- 第1页"行政复议申请材料接收清单" → 证据名称列表（4项）
- 第3页起"行政复议申请书" → 申请人信息

**从答复.pdf提取：**
- 第1页 → 被申请人收到答复通知书日期
- 落款页 → 作出答复日期（如文本层乱码，用vision.py OCR）

**从决定.pdf提取：**
- 第1页 → 被申请人全称、收到申请日期、受理日期
- 最后几页 → 决定书日期

### 3. 填充模板

使用python-docx修改模板：

```bash
pip3 install python-docx --break-system-packages
```

**填充规则：**

| 字段 | 规则 |
|------|------|
| 证据名称（证据1） | 严格按申请.pdf材料接收清单，不要用模板原文 |
| 被申请人名称 | **必须全称**：常熟市公安局XX派出所（见全称规则memory） |
| 被申请人指代 | 用"**被一**"，不用"被申请人" |
| 证据3-证明目的 | 格式："XX于X年X月X日作出答复并提交证据材料，符合《行政复议法》规定" |
| 证据3-页数 | "具体行政行为证据材料由被一提供"（不空着） |
| 提交人/日期 | 分开在不同单元格，不拼在一格 |
| 所有页数 | 留空（用户最终确认） |
| 字体 | 全局统一宋体10.5pt |

### 4. 日期映射

| 证明目的中的事件 | 日期来源 |
|------|------|
| 收到复议申请 | 决定.pdf第1页 |
| 受理 | 决定.pdf第1页 |
| 被一作出答复 | 答复.pdf落款（提取不到则用"收到答复通知书后在法定期限内作出答复"） |
| 作出复议决定 | 决定.pdf最后几页 |

### 5. 输出

保存到用户桌面：`提交证据目录清单_<案号>_<申请人>.docx`
用 `open -R` 揭示文件位置。

## 注意事项

- 行政机关必须全称，简称规则见 memory `reference_respondent_full_names`
- ov文件夹路径从memory `workflow_evidence_list_autofill` 获取
- 硅基流动vision.py用于OCR乱码页：`python3 ~/.claude/scripts/vision.py <图片路径> "提取文字"`
