# 网络案截图

在立案审查页面为网络申请案件批量截图案件信息和申请信息，保存到对应案件文件夹，最后打包。

## 执行步骤

### 1. 运行截图脚本
```bash
cd $HOME/hermes && $HOME/hermes/venv/bin/python3 screenshot_online_cases.py
```

脚本会自动：
- 登录复议平台（验证码用 ddddocr OCR 识别）
- 点击左侧"立案审查"菜单
- 从每个PDF中提取申请人姓名（夏秋涵、岳宗财等）
- 搜索申请人 → 点击"办理"（优先网络申请版本）
- 切换到"案件信息"标签页 (showpage 1)
- 隐藏立案审查行，截图当事人信息+复议事项 → 保存为"案件信息截图.png"
- 切换到"申请信息"标签页 (showpage 0)，截图整个页面 → 保存为"申请信息截图.png"
- 保存到 `/Volumes/ScannedDocuments/要新建/网络案/20260506网络案N/`

### 2. 打包压缩
```bash
cd "/Volumes/ScannedDocuments/要新建/网络案" && zip -r "YYYYMMDD网络案N件.zip" YYYYMMDD网络案*/
```
命名格式：`日期+网络案+数字+件`，如 `20260506网络案7件.zip`

## 关键文件
- 截图脚本：`$HOME/hermes/screenshot_online_cases.py`
- 案件目录：`/Volumes/ScannedDocuments/要新建/网络案/`

## 技术细节
- 案件信息页面(caseInfo.jsp)是单表格结构 `<table class="caseinfo">`
- 行class区分section：`caseinfo-head`(当事人信息)、`fyitems-head`(复议事项)、`caseAcceptence-head`(立案审查)
- 截图策略：隐藏 `caseAcceptence-head` 及之后的行 → `locator('body').screenshot()` → 恢复隐藏行
- 申请信息页面用 `locator('body').screenshot()` 截取全页
- Tab映射：showpage(0)=申请信息, showpage(1)=案件信息, showpage(3)=受理, showpage(9)=文书材料
- 嵌套iframe：主页 → listFrame(acceptencelist) → detailFrame(acceptence.jsp) → innerFrame(caseInfo.jsp)
- 申请人姓名从PDF提取，用PyPDF2的 `re.search(r'申请人[：:]\s*([^\s，,。.]+)', text)`
