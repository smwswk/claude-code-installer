# 网络案全流程

从复议平台完成网络案的完整处理流程：下载 → 办理 → 截图 → 打包 → 打印。步骤依次执行，每步完成后自动进入下一步。打包完成后询问用户是否打印。

## 执行步骤

### 第1步：下载案件材料
```bash
cd $HOME/hermes && $HOME/hermes/venv/bin/python3 download_online_materials.py
```
- 登录复议平台 → 进入"网申审核" → 从下到上逐个下载材料zip
- 命名为 `YYYYMMDD网络案序号.zip`，保存到 `/Volumes/ScannedDocuments/要新建/网络案/`
- 自动跳过已下载文件，失败可重跑
- **下载完成后脚本会自动解压 zip 到同名子目录**（截图/打印的前提）

### 第2步：办理案件（分配审查人）
```bash
cd $HOME/hermes && $HOME/hermes/venv/bin/python3 online_audit_process.py
```
- 进入"网申审核" → 从下到上逐个办理
- 分配审查人：第一=张莉亚(28166)，第二=金英(28167)
- 已办理的案件API会返回失败，可安全跳过

### 第3步：截图案件信息
```bash
cd $HOME/hermes && $HOME/hermes/venv/bin/python3 screenshot_online_cases.py
```
- **前置条件：每个 `YYYYMMDD网络案N/` 目录下已包含申请书 PDF 和附件 JPG**（第1步下载脚本已自动解压；如果是手动跑的需先 `unzip -o ZIP -d FOLDER/`）
- 进入"立案审查" → 从PDF提取申请人姓名 → 搜索 → 点击"办理"（优先网络申请版本）
- 截图案件信息（当事人信息+复议事项，隐藏立案审查行）→ `案件信息截图.png`
- 截图申请信息（整个页面）→ `申请信息截图.png`
- 保存到 `/Volumes/ScannedDocuments/要新建/网络案/YYYYMMDD网络案N/`

**注意**：运行前需更新 `screenshot_online_cases.py` 中的 `APPLICANTS` 列表，填入当天案件的文件夹名和申请人姓名。

### 第4步：打包压缩
```bash
cd "/Volumes/ScannedDocuments/要新建/网络案" && zip -r "YYYYMMDD网络案N件.zip" YYYYMMDD网络案*/
```
命名格式：`日期+网络案+数字+件`，如 `20260506网络案7件.zip`

## 关键文件
- 下载脚本：`$HOME/hermes/download_online_materials.py`
- 办理脚本：`$HOME/hermes/online_audit_process.py`
- 截图脚本：`$HOME/hermes/screenshot_online_cases.py`
- 案件目录：`/Volumes/ScannedDocuments/要新建/网络案/`

## 技术细节
- 登录：Playwright + ddddocr验证码识别，用户名 zhangliya
- iframe嵌套：主页 → listFrame → detailFrame → innerFrame
- Tab映射：showpage(0)=申请信息, showpage(1)=案件信息, showpage(3)=受理, showpage(9)=文书材料
- 截图策略：隐藏 `caseAcceptence-head` 及之后的行 → `locator('body').screenshot()` → 恢复
- 列表页：网申审核用 `onlineCaseAuditList` frame，立案审查用 `acceptencelist` frame

### 第5步：打印（询问用户）

打包完成后，**询问用户是否需要打印**。若用户确认，执行打印：

```bash
cd $HOME/hermes && $HOME/hermes/venv/bin/python3 print_online_cases.py
```

运行前确认脚本中的 `DATE` 变量为当天日期。

**Sanity check**：脚本会在打印每个案件前检查目录里是否有 PDF；如果只有 PNG 截图说明 zip 还没解压，会跳过并提示，不会盲打孤儿截图。

打印顺序（每个案件内部）：
1. 申请书PDF（UUID命名的PDF）
2. 普通PDF（非UUID命名的PDF）
3. 图片（jpg/jpeg/png，不含截图）
4. 截图（案件信息截图.png + 申请信息截图.png）

打印设置：单面、黑白、A4、80g纸，图片/截图保留白边（fit-to-page），打印机 M233sdn__537444_
