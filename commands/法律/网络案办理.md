# 网络案办理

从复议平台"网申审核"页面批量办理案件（分配立案审查人）。

## 执行步骤

### 1. 运行办理脚本
```bash
cd $HOME/hermes && $HOME/hermes/venv/bin/python3 online_audit_process.py
```

脚本会自动：
- 登录复议平台（验证码用 ddddocr OCR 识别）
- 点击左侧"网申审核"菜单
- 从列表页获取所有待办理案件（`a[title="办理"]`）
- **从下到上**逐个处理案件
- 对每个案件执行：
  1. 调用 `updateCaseStatus.action` 更新案件状态
  2. 调用 `saveCaseByOnline.action` 复制案件信息，获取新案件ID
  3. 在新标签页打开 `chooseAcceptor.jsp` 选择审查人
  4. 设置第一立案审查人=张莉亚，第二立案审查人=金英
  5. 调用 `commit()` 提交选择
  6. 调用 `/xzfy/chooseDealMan/updateCaseStatus.action` 保存分配
  7. 调用 `/xzfy/caseInfo/saveFixedRegister.action` 保存登记信息

### 2. 确认结果
检查输出统计（成功/失败数量），如有失败可重新运行（已办理的案件API会返回失败，可安全跳过）。

## 关键文件
- 办理脚本：`$HOME/hermes/online_audit_process.py`
- 保存目录：`$HOME/hermes/screenshots/`（调试截图）

## 技术细节
- 使用 Playwright + ddddocr 登录（与其他 hermes 脚本一致）
- 列表页 frame URL 含 `onlineCaseAuditList`
- chooseAcceptor.jsp 使用原生 HTML `<select>`（非 EasyUI combobox），需用 `selectedIndex` + `value` 设置
- 绕过 bsWindow 弹窗：在新标签页直接访问 chooseAcceptor.jsp URL
- 绕过 `$.MsgBox.Confirm` 确认框：直接调用 API（updateCaseStatus + saveCaseByOnline）
- 审查人选项值：张莉亚(28166)，金英(28167)
