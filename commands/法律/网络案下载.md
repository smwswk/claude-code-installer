# 网络案下载

从复议平台"网申审核"页面批量下载案件材料zip文件，保存到 `/Volumes/ScannedDocuments/要新建/网络案/`。

## 执行步骤

### 1. 运行下载脚本
```bash
cd $HOME/hermes && $HOME/hermes/venv/bin/python3 download_online_materials.py
```

脚本会自动：
- 登录复议平台（验证码用 ddddocr OCR 识别）
- 点击左侧"网申审核"菜单
- 从列表页获取所有案件的 caseId
- **从下到上**逐个处理案件
- 在新标签页直接访问材料页 URL（绕过 bsWindow 弹窗）
- 调用 `batchDownload()` 下载材料 zip
- 命名为 `YYYYMMDD网络案序号.zip`（如 `20260506网络案1.zip`）
- 保存到 `/Volumes/ScannedDocuments/要新建/网络案/`
- 自动跳过已存在的同名文件

### 2. 确认结果
检查输出统计（成功/失败数量），如有失败可重新运行（已下载的文件会自动跳过）。

## 关键文件
- 下载脚本：`$HOME/hermes/download_online_materials.py`
- 探索脚本：`$HOME/hermes/explore_online_audit.py`（用于发现页面结构，一般无需运行）
- 保存目录：`/Volumes/ScannedDocuments/要新建/网络案/`

## 技术细节
- 使用 Playwright + ddddocr 登录（与其他 hermes 脚本一致）
- 列表页 frame URL 含 `onlineCaseAuditList`
- 材料页 URL：`/xzfy/jsp/online/receiveBeforeShowFile.jsp?caseId=xxx`
- 下载通过新标签页直接访问材料页 + JS 调用 `batchDownload()`，避免 bsWindow 弹窗 iframe 加载问题
- 序号自动递增：扫描保存目录已有文件确定起始序号
