# 改编号

对已录入的案件执行立案审查 + 改编号流程。在系统里逐个搜索案件、点击办理、编辑表单、修正编号、保存。

## 执行步骤

### 1. 准备案件数据
从 `get_all_cases()` 获取案件列表（含第一批硬编码 + 第二批从Excel读取），计算正确编号：
- 不予受理案件：2位原号前加"10"（41→1041），3位原号前加"1"（100→1100）
- 正常复议案件：编号不变

### 2. 运行立案审查脚本
```bash
cd $HOME/hermes && $HOME/hermes/venv/bin/python3 case_review.py
```

脚本自动完成：
- 登录系统（验证码OCR识别）
- 导航到立案审查页面
- 逐个搜索申请人 → 点击办理 → 点击编辑 → 填写表单 → 保存
- 表单填写内容：行政管理类别→其他、处理类型→立案受理/不予受理、行政复议请求→撤销部分撤销行政行为并责令重作、案件编号序号修正、受理时间（仅受理案件）

### 3. 确认结果
检查脚本输出，确认每个案件保存成功。注意：
- 重名案件自动取第一条搜索结果，需手工核查是否正确
- 导航返回列表页偶尔失败，脚本会自动重试3次

## 关键文件
- 脚本：`$HOME/hermes/case_review.py`
- Excel：`$HOME/hermes/案件登记详细信息表.xlsx`
- 截图：`$HOME/hermes/screenshots/`

## 独立使用
也可以单独说"改编号"来只跑这一步（适用于之前录入过但没改编号的情况）。
