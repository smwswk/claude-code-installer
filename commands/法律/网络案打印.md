# 网络案打印

打印当天网络案文件夹的全部内容，以每个案件为单位按顺序打印。

## 执行

```bash
cd $HOME/hermes && $HOME/hermes/venv/bin/python3 print_online_cases.py
```

运行前需确认脚本中的 `DATE` 变量为当天日期。

## 打印顺序（每个案件内部）

1. **申请书PDF** — UUID命名的PDF（如 `fd4bc368-2218-49bc-bbaa-2db97348b0d4.pdf`）
2. **普通PDF** — 非UUID命名的PDF（如证据材料等）
3. **图片** — jpg/jpeg/png 文件（不含截图）
4. **截图** — `案件信息截图.png` + `申请信息截图.png`

## 打印设置

- 打印机：M233sdn__537444_（HP LaserJet M233sdn）
- 单面、黑白、A4、80g纸（stationery-bond）
- 图片和截图：fit-to-page（缩放至A4，保留白边，不适应边框）
- PDF：原尺寸打印

## 关键文件

- 打印脚本：`$HOME/hermes/print_online_cases.py`
- 案件目录：`/Volumes/ScannedDocuments/要新建/网络案/`
